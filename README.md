## 📖 Descripción

**Nich-Ká** es una aplicación web y móvil que permite a productores y procesadores de café monitorear y controlar de forma inteligente el proceso de fermentación del grano, apoyándose en inteligencia artificial para optimizar la calidad del producto final.

La plataforma registra variables clave del proceso —como temperatura, pH, tiempo y humedad— y utiliza modelos de AI para predecir el punto óptimo de fermentación, alertar sobre desviaciones y generar recomendaciones en tiempo real.

---

## ✨ Características principales

- 📊 **Dashboard en tiempo real** — visualización de variables del proceso de fermentación
- 🤖 **Control con AI** — predicciones y recomendaciones automáticas basadas en datos históricos
- 📱 **Multiplataforma** — experiencia nativa en web y móvil
- 🔔 **Alertas inteligentes** — notificaciones cuando el proceso se desvía de los parámetros óptimos
- 📈 **Historial de lotes** — registro y trazabilidad de cada proceso
- 🗺️ **Gestión de fincas** — organización por finca, lote y variedad de café

## 🐛 Issues y gestión de tareas

Usamos **GitHub Issues** para gestionar el trabajo de diseño. Puedes ver las tareas abiertas en la pestaña [Issues](../../issues).

### Etiquetas disponibles

| Etiqueta | Descripción |
|---|---|
| `design` | Tarea de diseño general |
| `new-screen` | Nueva pantalla o flujo |
| `bug` | Inconsistencia o error en el diseño |
| `review` | Listo para revisión |

## 📝 Notas de diseño

- El nombre **Nich-Ká** proviene del maya y significa *"Flor del agua"*, evocando el proceso de fermentación húmeda del café.
- La paleta de colores está inspirada en los tonos del café: tierras, verdes naturales y dorados.
- El diseño prioriza la usabilidad en campo, por lo que la versión móvil es la principal.
